# Roman to integer

From https://leetcode.com/problems/palindrome-linked-list/

## Description

Given the head of a singly linked list, return true if it is a palindrome.

Example 1:

```text
Input: head = [1,2,2,1]
Output: true
```

Example 2:
```text
Input: head = [1,2]
Output: false
```

Constraints:

- The number of nodes in the list is in the range [1, 10^5].
- 0 <= Node.val <= 9
